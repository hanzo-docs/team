## List and kanban views

The default view for issues is the list view, where issues are listed in order of their status. Hanzo Team also offers the option for a kanban view, which displays issues in a column format. To toggle between the two views, click either the list or kanban icon in the top right corner of the window.

**All Issues** in the navigator gathers issues from every project into one place, grouped by status (In Progress, Backlog, and so on). The toolbar across the top toggles the layout, grouping, and view settings, with `All` / `Active` / `Backlog` shortcuts and a search box:

![All Issues view listing issues across projects](/img/tracker/all-issues-desktop.png)

*All Issues, grouped by status. Each row shows the issue ID, title, estimate, due date, and assignee.*

> **On mobile:** the same grouped list scrolls in a single column; some columns are trimmed to fit.

![All Issues view listing issues across projects (mobile)](/img/tracker/all-issues-mobile.png)

*All Issues on a phone.*

When viewing issues for a specific project, you'll only be able to view one [task type](/docs/advanced-settings/task-types) at a time. This is because task types can have different process states. To view all of your tasks in one place (including all task types), navigate to `All issues` in the sidebar.

## Issue peek

Press the `space` bar while hovering over any issue to "peek" at that Issue's details. From this view, you can use the up and down arrows on your keyboard to scroll through your issues.

Hovering over comments or due dates will also open a preview of these details.

## Keyboard shortcut

Press `C` on your keyboard to create a new issue from any view.

## Sorting, filtering and searching

Options for sorting and filtering issues can also be found in the top right corner. The `View` button allows for configuration of some viewing settings as well as options for sorting how issues are grouped and ordered on the page. The `Show` button displays toggles for showing and hiding issue details. 

The `Filter` button, next to the search bar, offers options for filtering issues. The search bar can be used to search for a specific issue by keyword.

For quick sorting, select `Active`, `Backlog` or `All` from the menu above the search bar.
